This repository contains the code for the paper ‘Machine learning-guided hierarchical design of tandem catalysts for methyl methacrylate synthesis’. A dual Cu/Cs tandem catalyst has recently been identified for methyl methacrylate synthesis. However, its rational and efficient design remains challenging because of the inherent multiscale coupling in the tandem catalytic system. To address this issue, we propose a machine learning-enabled hierarchical strategy for the design of the Cu/Cs tandem catalyst.
CNN model: The ‘CNN model’ folder contains the data and code used for model construction and prediction. The CNN model is built from structural matrices to learn the relationship between spatial organization and acid-base properties. In particular, ‘Original_database’ provides the dataset used for the ‘Model construction’ code. ‘Cs-Si_61421’, ‘Ti-Cs_59917’, and ‘Ti-Si_61285’ are input datasets for the ‘Prediction’ code. The predicted high-performance catalyst features Ti–O–Cs structures with a Ti:Cs molar ratio of 1:1 and moderate spatial dispersion.
Neural ODEs model: The ‘Neural ODEs model’ folder contains the code used for catalyst bed design and product yield calculation. ‘model_params’ is utilized by the ‘Neural ODEs’ code. The optimized catalyst bed profile in terms of axial distribution of Cu and Cs catalysts can be obtained.
